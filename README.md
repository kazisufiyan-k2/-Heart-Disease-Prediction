# Heart Disease Prediction ❤️

This project predicts the risk of heart disease using Machine Learning and provides an interactive web interface using Streamlit.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* Jupyter Notebook

## Project Structure

* **Jupyter Notebook** – Model training and data analysis
* **Dataset** – Used for training and testing the model
* **Streamlit App** – Web interface for real-time prediction

## Features

* User-friendly web interface
* Real-time heart disease prediction
* Data preprocessing and scaling
* Machine learning model training in Jupyter Notebook

## How to Run the Project

1. Install dependencies

pip install -r requirements.txt

2. Run the Streamlit app

streamlit run app.py
