# Heart Disease Prediction ❤️

This project predicts the risk of heart disease using Machine Learning and provides a web-based interface built with Flask and custom HTML/CSS.
## Live Demo
https://heart-disease-prediction-1-skte.onrender.com

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Flask
* HTML
* CSS
* Jupyter Notebook

## Project Structure

* **Jupyter Notebook** – Model training and data analysis
* **Dataset** – Used for training and testing the model
* **Flask App (app.py)** – Backend server and routing
* **Templates (HTML)** – Frontend structure
* **Static (CSS)** – Styling and UI design

## Features

* Clean and responsive web interface
* Real-time heart disease prediction
* Data preprocessing and scaling
* Machine learning model trained using Jupyter Notebook
* Separation of frontend and backend

## How to Run the Project

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the Flask app

```
python app.py
```

3. Open in browser

```
http://127.0.0.1:5000/
```
